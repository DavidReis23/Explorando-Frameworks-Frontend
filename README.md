# Projeto Site Pessoal/Blog - React

## Frameworks Utilizados
- **React 18.2.0** - Biblioteca principal para construção da interface
- **React Router DOM 6.20.0** - Para navegação entre páginas
- **Vite 7.3.1** - Ferramenta de build e servidor de desenvolvimento

## Passo a Passo para Rodar Localmente

1. **Clone o repositório** ou use os arquivos do projeto
2. **Abra o terminal** na pasta do projeto (`blogDavid`)
3. **Instale as dependências:**
   ```bash
   npm install
   ```
4. **Execute a aplicação:**
   ```bash
   npm run dev
   ```
5. **Acesse no navegador:**
   - **Local:** http://localhost:5173
   - A aplicação atualiza automaticamente ao salvar mudanças

## Principais Dificuldades Encontradas

1. **Componentização** - Decidir quais partes se tornariam componentes reutilizáveis
2. **Gerenciamento de rotas** - Configurar o React Router para navegação entre páginas
3. **Estilização responsiva** - Garantir que o layout funcionasse em diferentes tamanhos de tela
4. **Organização do código** - Estruturar os arquivos de forma clara e mantenível

## Semelhanças e Diferenças entre Frameworks

*Nota: Como utilizei apenas React, posso comparar com conhecimento teórico de Vue.js:*

### Semelhanças:
- Ambos usam **componentização** para criar interfaces
- Possuem **sistemas de rotas** para navegação (React Router / Vue Router)
- Usam **conceito de estado reativo** para atualizar a UI
- São **frameworks baseados em componentes**

### Diferenças:
- **React usa JSX** (JavaScript + HTML), **Vue usa templates** (HTML-like)
- **React** é uma biblioteca (precisa de mais decisões), **Vue** é um framework (mais opinativo)
- **React** gerencia estado com Hooks, **Vue** tem sistema reativo nativo
- **Vue** tem Single File Components (.vue), **React** separa em arquivos diferentes
